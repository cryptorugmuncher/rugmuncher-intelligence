## Design Specifications for PDF Export

### Layout
- **Orientation**: Landscape (11" × 8.5")
- **Margins**: 0.5" on all sides
- **Column Grid**: 3-column layout for top section, full-width sections below
- **Typography**:
  - Headings: Inter Bold, 14-16pt
  - Body: Inter Regular, 9-10pt
  - Callouts/numbers: Inter SemiBold, 11pt

### Color Palette
- **Header Background**: Deep Purple (#2E1A47)
- **Section Headers**: Brand Orange (#FF6B35)
- **Accent**: Electric Blue (#00D9FF)
- **Body Text**: Dark Gray (#1A1A1A)
- **Background**: White (#FFFFFF)

### Visual Elements
- **Logo**: Top center, 60px height
- **QR Code**: Bottom right, 80px × 80px (links to investor portal)
- **Boxes**: 1px solid borders, 8px border radius, subtle drop shadows
- **Icons**: Simple line icons (money, users, rocket, shield)

### Typography Hierarchy
```
H1 (Page Title):        Inter Bold, 18pt, Deep Purple
H2 (Section Headers):   Inter SemiBold, 14pt, Brand Orange
H3 (Subsections):       Inter SemiBold, 11pt, Dark Gray
Body Text:              Inter Regular, 9pt, Dark Gray
Numbers/Stats:          Inter Bold, 12-14pt, Electric Blue
Small Print/Footer:     Inter Regular, 8pt, Medium Gray
```

### Content Density Guidelines
- **Line height**: 1.4 (for readability)
- **Paragraph spacing**: 6pt between paragraphs
- **Section spacing**: 12pt between major sections
- **Bulleted lists**: 4pt spacing between items
- **Tables**: Alternating row backgrounds (white, light gray #F8F8F8)

### Export Instructions

**Option 1: Markdown → Pandoc → PDF**
```bash
pandoc one-pager.md -o one-pager.pdf \
  --pdf-engine=xelatex \
  --variable geometry:"margin=0.5in" \
  --variable fontfamily:"Inter" \
  --variable fontsize:9pt \
  --variable papersize:"letter" \
  --variable orientation:"landscape"
```

**Option 2: Figma Design**
1. Create artboard: 1100px × 850px (11" × 8.5" at 100 DPI)
2. Import content from this markdown
3. Apply brand colors and typography
4. Export as PDF (print quality, 300 DPI)

**Option 3: Google Docs → PDF**
1. Set page to Landscape orientation
2. Set margins to 0.5" on all sides
3. Use Inter font family (import from Google Fonts)
4. Apply colors manually (#2E1A47, #FF6B35, #00D9FF)
5. Export as PDF
