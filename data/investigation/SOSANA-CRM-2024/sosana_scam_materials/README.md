# CryptoRugMunch Visual Assets

This directory contains all visual materials for CryptoRugMunch including logos, banners, diagrams, pitch decks, and whitepapers.

## Directory Structure

### `/logo` - Brand Logos
- `logo.svg` - Vector logo (scalable)
- `logo.png` - 1024x1024 PNG for general use
- `logo-icon.svg` - Icon only (shield with bite mark)
- `logo-icon.png` - 512x512 PNG icon
- `logo-horizontal.svg` - Wordmark + icon horizontal layout

### `/banners` - Social Media Banners
- `github-banner.png` - 1280x640 repository banner
- `twitter-banner.png` - 1500x500 Twitter header
- `telegram-banner.png` - Telegram channel cover

### `/diagrams` - Technical & Business Diagrams
- `architecture.png` - System architecture diagram
- `risk-algorithm.png` - 12-metric risk scoring breakdown
- `tokenomics.png` - Token supply and utility infographic
- `roadmap.png` - 18-month timeline
- `revenue-model.png` - Business model infographic
- `growth-projections.png` - 3-year financial projections

### `/pitch-decks` - Presentation Materials
- `investor-pitch.pdf` - 12-15 slide investor deck
- `one-pager.pdf` - Single-page executive summary
- `team-whale-deck.pdf` - 20-25 slide detailed technical deck

### `/whitepapers` - Technical Documentation
- `technical-whitepaper.pdf` - 15-20 page technical specification
- `economic-whitepaper.pdf` - 12-15 page token economics
- `product-whitepaper.pdf` - 10-12 page product overview

## Brand Guidelines

**Colors**:
- Brand Orange: #FF6B35
- Deep Purple: #2E1A47
- Electric Blue: #00D9FF
- Safe Green: #00C48C
- Caution Yellow: #FFB800
- Danger Red: #FF3B3B

**Typography**:
- Primary: Inter (all weights)
- Code/Technical: JetBrains Mono

**Export Quality**:
- Logos: SVG (vector) + PNG at 300 DPI
- Diagrams: PNG at 300 DPI for print, 72 DPI for web
- Documents: PDF format with embedded fonts

See: [Brand Guidelines](../docs/04-GTM/brand-guidelines.md) for complete specifications.
