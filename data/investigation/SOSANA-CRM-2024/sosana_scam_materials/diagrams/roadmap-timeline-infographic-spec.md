## Design Tools & Templates

### Figma Template
```
1. Create artboard: 1600px × 900px (16:9 ratio)
2. Import brand colors: #2E1A47, #FF6B35, #00D9FF, #00C48C, #FFB800
3. Import Inter font family from Google Fonts
4. Use Auto Layout for timeline boxes (consistent spacing)
5. Create components:
   - Timeline dot (40px circle, reusable)
   - Deliverable box (white with colored left border)
   - Metric box (light gray background)
   - Milestone call-out (small floating box)
6. Export: PNG (2x), PDF (print quality), SVG (web)
```

### Canva Template
```
1. Search for "Timeline Infographic" templates
2. Customize colors to match brand palette
3. Use built-in chart tools for MAU/MRR growth chart
4. Replace placeholder text with roadmap milestones
5. Export: PDF (high quality) or PNG (4x resolution)
```

### Adobe Illustrator
```
1. New document: 11\" × 8.5\" landscape, CMYK mode
2. Create timeline path: Pen tool, 3px stroke
3. Add milestone dots: Circle tool, 40px diameter
4. Create text boxes for deliverables/metrics
5. Use Data Graph for MAU/MRR chart
6. Export: PDF/X-1a (print) or PNG 300 DPI
```
