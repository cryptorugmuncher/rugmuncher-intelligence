## Design Tools & Templates

### Figma Template
```
1. Create artboard: 800px × 1200px (1:1.5 aspect ratio)
2. Import brand colors: #2E1A47, #FF6B35, #00D9FF, #00C48C
3. Import Inter font family from Google Fonts
4. Create components:
   - Token stat card
   - Utility quadrant
   - Chart template
5. Use Figma's pie chart plugin for allocation chart
6. Export: PNG (2x), PDF (print quality)
```

### Canva Template
```
1. Search for "Infographic" templates
2. Customize colors to match brand palette
3. Use built-in chart tools for pie chart and area chart
4. Import custom icons from Flaticon or Noun Project
5. Export: PDF (high quality) or PNG (4x resolution)
```

### Adobe Illustrator
```
1. New document: 8.5" × 11", CMYK mode
2. Create artboards for each section
3. Use Data-driven charts for pie chart and area chart
4. Export: PDF/X-1a (print) or PNG 300 DPI
```
