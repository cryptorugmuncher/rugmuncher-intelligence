# Logo Assets

## Required Files

1. **logo.svg** - Full vector logo
   - Format: SVG
   - Contains: Shield icon + "CryptoRugMunch" wordmark
   - Colors: Brand Orange (#FF6B35) + Deep Purple (#2E1A47)
   - Use: Scalable applications, print materials

2. **logo.png** - High-resolution raster logo
   - Format: PNG (transparent background)
   - Size: 1024x1024px @ 300 DPI
   - Use: Social media, presentations

3. **logo-icon.svg** - Icon only (no text)
   - Format: SVG
   - Contains: Shield with bite mark icon
   - Size: Square aspect ratio
   - Use: Favicons, app icons, small spaces

4. **logo-icon.png** - Icon raster version
   - Format: PNG (transparent background)
   - Size: 512x512px @ 300 DPI
   - Use: Social media avatars, GitHub profile

5. **logo-horizontal.svg** - Horizontal layout
   - Format: SVG
   - Contains: Icon + wordmark side-by-side
   - Use: Headers, navigation bars

## Design Specifications

**Shield Icon**:
- Represents protection/security
- Bite mark detail (references "rug muncher" name)
- Clean, modern geometric design
- Works well at small sizes (16x16px minimum)

**Wordmark**:
- Font: Inter Bold
- Case: Mixed (CryptoRugMunch)
- Letter spacing: -0.02em
- Color: Deep Purple (#2E1A47) or white (on dark backgrounds)

**Color Variants**:
- Primary: Orange shield + purple text
- Reversed: White on dark backgrounds
- Monochrome: All white or all purple for special cases
